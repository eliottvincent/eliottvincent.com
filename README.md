# eliottvincent.com
My personal website. [[view live]](https://eliottvincent.com)


#### Generate a development build
```sh
./tasks/start.sh
```

#### Generate a production build
```sh
./tasks/start.sh production
```

#### Run in development mode (with hot-reload)
```sh
./tasks/dev.sh
```

#### Update dependencies
```sh
bundle update
```
